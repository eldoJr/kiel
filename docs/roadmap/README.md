# Build Order (Practical)

```
v0 → v1 → v2 → v3 → v4 → v5 → v6
```

## Phase Progression

| Phase | Focus | Description |
|-------|-------|-------------|
| **v0** | Brain | Core cognitive intelligence |
| **v1** | Actions | Execution engine |
| **v2** | Agents | Specialized agent system |
| **v3** | Memory | Long-term context & persistence |
| **v4** | Voice | Speech interface & presence |
| **v5** | System | OS-level integration |
| **v6** | Proactive | Autonomous suggestions |
